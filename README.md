# Sad Course Bot

Telegram Bot on NestJS for parse and notificate exchange rates from [Minfin](https://minfin.com.ua/currency/mb/).

## Stack

1. [NestJS](https://nestjs.com/)
* [Configuration](https://docs.nestjs.com/techniques/configuration)
* [Task Sheduling](https://docs.nestjs.com/techniques/task-scheduling)
* [Connect to DB](https://docs.nestjs.com/techniques/mongodb)
* [Validation Configuration](https://docs.nestjs.com/techniques/validation)
* [Logger](https://docs.nestjs.com/techniques/logger)
2. [Telegraf](https://github.com/telegraf/telegraf)
3. [Mongoose](https://mongoosejs.com/)
4. [Cheerio (Parser)](https://github.com/cheeriojs/cheerio)
